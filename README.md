Feedstock for [nnlojet](https://nnlojet.hepforge.org) — part of [hep-forge](https://anaconda.org/hep-forge).

A parton-level event generator for jet cross sections at NNLO QCD accuracy

## Install

```bash
conda install -c hep-forge -c conda-forge nnlojet
```
